# GitHub Image Compare

GitHub Enterprise 이슈 및 Pull Request에 첨부된 이미지를 브라우징하고 비교하는 Chrome 확장 프로그램입니다.

이슈의 이미지를 클릭하면 전체 캐로샐이 열리고, 키보드 화살표로 탐색하고, 두 이미지를 선택해 나란히 비교하고, 정량적인 유사도 지표를 확인할 수 있습니다 — 페이지를 벗어날 필요 없이.

[English README](./README.md)

## 기능

- **이미지 캐로샐** — 이슈/PR의 이미지를 클릭하면 페이지의 모든 이미지를 썸네일 스트립으로 보여주는 오버레이가 열립니다
- **키보드 탐색** — `←` / `→` 키로 이미지 이동, `Esc`로 닫기
- **비교 모드** — 썸네일 스트립에서 두 이미지를 선택하면 나란히 비교합니다
- **유사도 지표** — 외부 API 없이 브라우저 내에서 5가지 정량 지표를 계산합니다:

| 지표 | 설명 | 범위 |
|------|------|------|
| **SSIM** | 구조적 유사도 (인간 시각 모델 기반) | 0 – 1 (1 = 동일) |
| **MSE** | 평균 제곱 오차 (픽셀 수준 차이) | 0 – ∞ (0 = 동일) |
| **PSNR** | 신호 대 잡음비 | dB (높을수록 유사) |
| **pHash** | 지각적 해시 유사도 (DCT 기반) | 0 – 100% |
| **히스토그램 상관** | RGB 색상 분포 유사도 | −1 – 1 (1 = 동일) |

- **차이 히트맵** — 두 이미지의 픽셀 차이를 파랑(유사)~빨강(차이) 히트맵으로 시각화
- **Shadow DOM 격리** — GitHub의 CSS와 완전히 격리되어 스타일 충돌 없음
- **GitHub Enterprise 호환** — 도메인에 관계없이 모든 GHE 설치 환경에서 동작

## 설치

### 사전 요구사항

- Node.js 18 이상
- Chrome 110 이상 (또는 Chromium 기반 브라우저)

### 소스에서 빌드

```bash
git clone https://github.com/your-org/image-compare.git
cd image-compare
npm install
npm run build        # dist/ 폴더에 결과물 생성
```

플레이스홀더 아이콘 생성 (최초 1회만):

```bash
npm run gen-icons
```

### Chrome에 설치

1. `chrome://extensions` 접속
2. 우상단 **개발자 모드** 활성화
3. **압축해제된 확장 프로그램 로드** 클릭
4. `dist/` 폴더 선택

확장 아이콘이 도구 모음에 나타납니다. 최초 설치 시 기본 설정이 자동으로 저장됩니다.

### 개발 모드 (watch)

```bash
npm run dev     # 파일 변경 시 자동 리빌드
```

리빌드 후 `chrome://extensions`의 확장 카드에서 **↺ 새로고침** 버튼을 클릭하세요.

## 사용법

### 캐로샐 열기

GitHub 이슈 또는 PR의 본문이나 댓글에 있는 이미지를 클릭합니다. 오버레이가 열리면:

- 상단에 페이지의 모든 이미지가 **썸네일 스트립**으로 표시됩니다
- 클릭한 이미지가 **메인 영역**에 크게 표시됩니다
- 헤더에 이미지 인덱스와 파일명이 표시됩니다

### 탐색하기

| 동작 | 설명 |
|------|------|
| `←` / `→` 키 | 이전 / 다음 이미지 |
| 썸네일 클릭 | 해당 이미지로 바로 이동 |
| `Esc` | 오버레이 닫기 |
| 배경 클릭 | 오버레이 닫기 |

### 두 이미지 비교하기

1. 캐로샐 헤더의 **비교 모드** 버튼 클릭 → 선택 모드 진입
2. 썸네일 두 개를 클릭하면 **#1**, **#2**로 표시됩니다
3. **비교하기 →** 버튼 클릭 → 비교 화면 진입

비교 화면에서:

- 두 이미지가 좌우로 나란히 표시됩니다
- 5가지 유사도 지표가 자동으로 계산됩니다 (약 1초 소요)
- **차이 히트맵 보기** 버튼으로 픽셀 차이 시각화를 확인할 수 있습니다
  - 파랑 = 유사한 영역, 빨강 = 다른 영역

### 팝업 설정

도구 모음의 확장 아이콘을 클릭하면 설정 팝업이 열립니다:

| 설정 | 설명 |
|------|------|
| **확장 활성화** | 확장 전역 활성화 / 비활성화 |
| **GitHub Enterprise 도메인** | 선택 사항: 특정 GHE 도메인에서만 활성화 |

## 유사도 지표 계산 방법

모든 계산은 브라우저의 Canvas API를 사용하여 인-브라우저에서 처리됩니다. 이미지가 외부 서버로 전송되지 않습니다.

MSE, PSNR, SSIM, 히스토그램 상관은 두 이미지를 **256 × 256**으로 정규화한 후 계산합니다. pHash는 이미지를 **32 × 32**로 축소한 후 2D DCT를 적용합니다.

**SSIM**은 휘도 채널을 기준으로 **8 × 8** 비중첩 윈도우의 평균값으로 계산됩니다.

**pHash**는 32 × 32 그레이스케일 이미지에 2D DCT를 적용하고, 좌상단 8 × 8 계수(64비트)를 기준으로 두 해시의 Hamming 거리를 측정합니다.

## 기술 스택

| 영역 | 선택 |
|------|------|
| UI 프레임워크 | React 18 + TypeScript |
| 빌드 도구 | Vite 5 + `@crxjs/vite-plugin` |
| 스타일링 | Tailwind CSS 3 (Shadow DOM에 주입) |
| 상태 관리 | Zustand 4 |
| 이미지 처리 | Canvas API (브라우저 내장) |
| 확장 Manifest | MV3 |
